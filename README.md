# Silver Surfer Strength

Бесплатная платформа для силовых тренировок с весом тела

## Программы тренировок:
- Street Strength (уличные тренировки)
- Home Strength (домашние тренировки) 
- Hybrid Strength (смешанная программа)

## Особенности:
- 90-дневные программы
- Отслеживание прогресса
- Система достижений
- Полностью бесплатно
